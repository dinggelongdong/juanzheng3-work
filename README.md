## 功能介绍
### 系统总体结构设计

![gyf](C:\Users\Administrator\Pictures\gyf.png)

### 管理员端功能
- 查看用户列表
- 查看、修改、删除待审核的物资
- 查看待用户捐赠的物资
- 查看未通过审核的物资
- 查看捐赠记录
- 信息查看、信息修改、密码修改
### 用户端功能
- 用户登录
- 查看物资捐赠信息
- 捐赠物资
- 查看物资捐赠记录
- 查看待审核的物资
- 查看已通过审核的物资
- 查看未通过审核的物资
- 信息查看、信息修改、密码修改
